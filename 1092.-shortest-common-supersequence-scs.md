# 1092. Shortest Common Supersequence \(SCS\)

Adaptive to **LCS**

```cpp
len of SCS = m + n - len(LCS)
Printing SCS = Printing LCS + modify + print leftovers
```

First we do normal LCS dp array.

```cpp
int dp[n+1][m+1];
for(int i = 0; i <= n; i++){
        for(int j = 0; j <= m; j++){
            if(i == 0 || j == 0)
                dp[i][j] = 0;
            else if(X[i - 1] == Y[j - 1])
                dp[i][j] = 1 + dp[i - 1][j - 1];
            else
                dp[i][j] = max(dp[i][j - 1], dp[i - 1][j]);
        }
}
```

Then, we print it judiciously

```cpp
// dp[n+1][m+1], X, Y, n, m
int i = n, j = m;
string ans = "";
while(i > 0 and j > 0){
    if(X[i-1] == Y[j-1]){
        ans += X[i-1];
        i--;
        j--;
    }else if(dp[i][j-1] > dp[i-1][j]){
        ans += Y[j-1];// if we go left, take the Y char 
                        // and then move left
        j--;
    }else {
        ans += X[i-1];
        i--;
    }
}

// leftovers
while(i > 0){
    ans += X[i-1];
    i--;
}

while(j > 0){
    ans += Y[j - 1];
    j--;
}

// reverse bcoz we are traversing from the back
reverse(ans.begin(), ans.end());
```
